# GZ农商银行 MGM 裂变营销 H5

> 财富客户推荐有礼活动 — 微信内嵌 H5 原型

## 项目简介

GZ农商银行私钻客户（资产门槛 300W+）MGM 裂变营销活动 H5 页面，包含活动规则展示、在线报名、推荐码分享、达标进度查询、礼品领取等核心功能。

## 功能模块

| 模块 | 描述 |
|------|------|
| **活动首页** | 活动规则展示、报名入口 |
| **报名登记** | 姓名+手机号验证、推荐码生成 |
| **我的推荐** | 推荐码展示、二维码分享、推荐记录 |
| **进度/领取** | 礼品领取状态、额度展示 |

## 技术栈

- **前端**: 原生 HTML + CSS + JavaScript（无框架依赖）
- **适配**: 移动端优先，微信内嵌 H5（iOS / Android）
- **设计**: 银行金融风格，深蓝色主色调

## 快速预览

```bash
# 直接用浏览器打开
open index.html

# 或用 http-server 启动本地服务
npx http-server -p 8080
```

## 页面预览

### 1. 活动首页
- 活动 Banner + 规则卡片
- CTA 引导报名

### 2. 报名登记
- 表单验证（姓名、手机号、验证码）
- 报名成功自动生成推荐码

### 3. 我的推荐
- 推荐码 + 二维码展示
- 分享好友
- 推荐记录列表

### 4. 礼品领取
- 已领/上限额度展示
- 达标进度条
- 领取按钮

## 测试说明

- 验证码任意输入 `123456` 即可通过
- 礼品领取点击「立即领取」即可触发领取成功
